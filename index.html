Steamitemtracker

Track the location and item ID of steam items using steam's developer API.

Download as .zip Download as .tar.gz View on GitHub

Find your Steam, Dota2, CS:GO, and TF2 items

Track items you've traded to find item ID and location with a user friendly GUI that implements steam's developer API to give users easy access to their steam trading data.

Open Source

Open source means anyone is able to view and modify the code. Basically, anyone is able to contribute to the project!

User friendly and quick

You don't need any extensive knowledge of Steam's API to use this extension! A feature-extensive GUI with a developer console included gives you the most flexibility.

Safe and Secure

The extension doesn't store any of your trade data, and directly communicates with steam's developer API. The extension works inside the boundaries of steam, giving you an easier and secure way to access trade information without complexity.
